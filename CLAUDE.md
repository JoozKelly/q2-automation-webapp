# Q2 Economic Report Automation Web App

## Project Context
This project automates the quarterly economic reporting workflow for Batam tenants.
The core goal is to replace manual Excel charting and PPT design with an automated web application that:
1. Ingests CSV data from Genspark.ai.
2. Visualizes key macroeconomic indicators (GDP, investment, inflation) using Recharts/Chart.js.
3. Provides a Report Builder interface to integrate AI-generated storylines.
4. Exports a beautifully styled PDF mimicking the previous PPT templates.

## Workflow Rules & Commands
- **Git Commits:** Always commit work regularly with clear, descriptive commit messages to ensure we never lose status or work. Push to GitHub when the repository is linked.
- **Code Structure:** Built using Next.js, Recharts, and TailwindCSS (for precise styling mirroring the PPT template).
- **Automation:** Utilize MCPs or CLI tools as needed to enhance data ingestion and copywriting.
